SETUP — Hearth Variations

Three SVG centerpiece options inside the same Hearth shell. The rest of the screen (rounds grid, embers, header, footer) is identical across all three so you can compare just the flame element apples-to-apples.

01

Bonfire

Cluster of 6-7 smaller flames at the base, each flickering independently. Like a campfire view. More organic — fire is a group of flames, not one big shape.

02

Halo

Drop flames entirely. Radial sunburst — rotating light spokes plus concentric pulse rings. More abstract / mythic. A "sun of heat" instead of a literal fire.

03

Turbulent

Single big flame with SVG turbulence filter applied for organic wavering edges. Same composition as the original Hearth but the flame actually looks like a flame.
