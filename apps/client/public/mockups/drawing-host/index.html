Drawing Host (Tablet) — Mockups

Three takes on the Samsung-tablet host surface for the DRAWING minigame. All show mid-drawing state — prompt visible, a few strokes in, palette and brush selected, Correct/Incorrect/Skip/Undo/Clear all reachable. Built for ~16:10 landscape.

Open in a browser at roughly 1600×1000 (or full-screen) for a tablet-accurate preview. All three share the Spicy Broadcast tokens.

01

Sketchpad Studio

Warm cream canvas inside the dark Spicy Broadcast chrome. Tools laid out like a real art tray — paint-dab palette, brush silhouettes, chunky tactile buttons. Handcrafted and inviting.

02

Heat Console

Pure dark with neon-orange stroke glow on a matte graph-paper canvas. Industrial control-dock toolbar with LED-style color pucks. Drawer feels like they're at a mission console.

03

Showtime Easel

Game-show theatrics. Marquee prompt banner up top, framed chalkboard easel canvas, illuminated palette lights along the footer. Most playful and characterful.
